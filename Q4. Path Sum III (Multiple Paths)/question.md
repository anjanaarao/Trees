**Difficulty:** Medium

**Issue Description:**  
Count the number of paths that sum to a target value (paths need not start at root).  
This is a brainstorming problem requiring prefix-sum logic on trees.

**Acceptance Criteria:**  
- Correct path count

- Time Complexity: O(n)  
- Space Complexity: O(n)
