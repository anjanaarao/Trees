## 11: Recover Binary Search Tree

**Difficulty:** Hard  
**Tags:** bst, inorder  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/recover-binary-search-tree/  

**Concept to Read:**  
- https://www.geeksforgeeks.org/fix-two-swapped-nodes-of-bst/  

**Issue Description:**  
Two nodes of a BST are swapped. Restore the BST without changing structure.  
Promotes anomaly detection using inorder traversal.

**Acceptance Criteria:**  
- Correct BST recovery  
- Constant extra space preferred  
- Time Complexity: O(n)  
- Space Complexity: O(h)
