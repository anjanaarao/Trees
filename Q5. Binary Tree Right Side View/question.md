## 5: Binary Tree Right Side View

**Difficulty:** Medium  
**Tags:** binary-tree, bfs  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/binary-tree-right-side-view/  

**Concept to Read:**

**Issue Description:**

Requires careful ordering in traversal.

**Acceptance Criteria:**  
- Correct right view  
- Handles skewed trees  
- Time Complexity: O(n)  
- Space Complexity: O(n)
