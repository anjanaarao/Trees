## 13: Flatten Binary Tree to Linked List

**Difficulty:** Medium-Hard 
**Tags:** binary-tree, recursion  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/flatten-binary-tree-to-linked-list/  

**Concept to Read:**

**Issue Description:**  
Transform the tree into a right-skewed linked list in-place.  
Tests pointer manipulation and traversal order.

**Acceptance Criteria:**  
- Correct flattened structure  
- No extra nodes created  
- Time Complexity: O(n)  
- Space Complexity: O(h)
