## 12: Maximum Sum BST in Binary Tree

**Difficulty:** Hard  
**Tags:** binary-tree, bst  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/maximum-sum-bst-in-binary-tree/  

**Concept to Read:**  
- https://www.geeksforgeeks.org/largest-sum-bst-in-binary-tree/  

**Issue Description:**  
Find the maximum sum of values of any subtree that is a valid BST.  
Combines validation + aggregation.

**Acceptance Criteria:**  
- Correct maximum sum  
- Handles negative values  
- Time Complexity: O(n)  
- Space Complexity: O(h)
