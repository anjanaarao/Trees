## 7: All Nodes Distance K in Binary Tree

**Difficulty:** Medium  
**Tags:** binary-tree, bfs, dfs  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/all-nodes-distance-k-in-binary-tree/  

**Concept to Read:**  
- https://www.geeksforgeeks.org/print-nodes-distance-k-given-node-binary-tree/  

**Issue Description:**  
Return all nodes at distance K from a target node.  
This problem forces thinking beyond downward traversal.

**Acceptance Criteria:**  
- Correct node list  
- Handles parent traversal  
- Time Complexity: O(n)  
- Space Complexity: O(n)
