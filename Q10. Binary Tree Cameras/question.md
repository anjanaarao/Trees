## 10: Binary Tree Cameras

**Difficulty:** Hard  
**Tags:** binary-tree, greedy  

**Problem Reference:**  
- Platform: LeetCode  
- Link: https://leetcode.com/problems/binary-tree-cameras/  

**Concept to Read:**  
- https://www.geeksforgeeks.org/vertex-cover-problem-set-1-introduction/  

**Issue Description:**  
Place minimum cameras so every node is monitored.  
This problem requires greedy state-based recursion.

**Acceptance Criteria:**  
- Correct minimum camera count  
- Handles leaf-heavy trees  
- Time Complexity: O(n)  
- Space Complexity: O(h)
